# 成果物搬入カード

**成果物搬入カード**は**パターンプロバイダー**専用のアップグレードであり、バニラのかまどなどの自動搬出ができないブロックにインポートバスが不要になるため、自動化する設定を簡単にします。

このカードを挿入すると、**プロバイダー自身**が隣接するブロックから成果物を搬入することが可能になります。つまり、これで「搬出」と「搬入」の動作を一つのプロバイダーにまとめることができます。

また、プロバイダーにこのカードを挿入する専用のスロットが追加されます。

## 設定
このカードを持って右クリックすると、設定画面が開きます:

*   **成果物のみ搬入**:
    *   **有効 (デフォルト)**: このカードは「賢い」です。搬入された素材を確認し、予想される出力を計算します。そして、その出力のみを搬入し、他のアイテムは無視します。
    *   **無効**: このカードは、普通のインポートバスと同じ動きをします。対象のブロックからあらゆるアイテムをネットワークに搬入します。
    *   有効にすると、ブロックから常に搬入を試みる必要がなくなるため、パフォーマンスが大幅に向上します。

*   **クラフティングCPUと同期**:
    *   **有効 (デフォルト):** このカードは、自動クラフトを管理するクラフティングCPUと通信しています。自動クラフトをキャンセルすると、カードが成果物の搬入を止めます。これによって、プロバイダーが決して届かないアイテムを待ち続けることが無くなります。

*   **搬入面**:
    *   **自動**: プロバイダーが接している面から搬入します。
    *   **面指定 (上、下、北…):** プロバイダーが、隣接するブロックの特定の面から搬入するように指定します。バニラのかまどのように、特定の面からのみ出力が可能なブロックに有用です。


このカードは、搬入する頻度が少ない場合、サーバーの負荷を抑えるためにインベントリをチェックする頻度を下げます。新しいクラフトが開始されると、動作が元に戻ります。

### 制限
*   **方向指定プロバイダー専用**: このカードは、パネル化したパターンプロバイダーと方向を指定したフルブロックのパターンプロバイダーで使用することができます。フルブロックの方向指定については、レンチを使って設定可能です。
