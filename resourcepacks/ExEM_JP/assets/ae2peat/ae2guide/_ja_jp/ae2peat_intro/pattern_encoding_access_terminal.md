# MEパターンエンコーディングアクセスターミナル (P. E. A. ターミナル)

MEパターンエンコーディングアクセスターミナルは、パターンのエンコーディング機能とアクセス機能を統合したターミナルです。
MEネットワークに接続するだけで、1つの画面でクラフト、加工、鍛冶、石切のパターンを作成・編集・閲覧できます。
