# 世界加速装置

**世界加速装置**は、MEネットワークから電力を消費して、前方のブロックの動きを加速させる装置です。言うなれば「Torcherino」、「時流計」です。

<GameScene zoom="8" background="transparent">
<ImportStructure src="../../structure/entity_speed_ticker.snbt"></ImportStructure>
</GameScene>

## 仕組み

1. **有効化の条件**: 加速を有効にするには、少なくとも装置に一つの **<ItemLink id="extendedae_plus:entity_speed_card" tag="{'EAS:mult':2}" />** かそれ以上の倍率の世界加速カードが入っている必要があります。
2. **加速率**: **1024倍**まで加速できます。
3. **電力消費**: 加速中は、MEネットワークから電力を消費し続けます。電力が不足すると、回復するまで加速は一時停止します。
    - もし*Applied Flux*が導入されていれば、AEの電力が不足したときにディスクにあるForge Energyを消費して加速を継続させることができます。消費の優先順位は変更できます。
4. **コンフィグ設定**: 基礎エネルギー消費量、加速対象のブラックリスト、加速によるエネルギー乗数は、コンフィグから設定可能です。

## エネルギー消費の計算

### 基礎消費量の計算

世界加速装置の電力消費量は、コンフィグの`baseCost`と加速率によって決まります。

定義:
- \( M \) = 加速率 (2, 4, 8, ..., 1024)
- \( B \) = コンフィグの基礎消費量 (デフォルトでは512)
- \( L \) = log₂(M)

**基礎消費量の計算式**:

![eformula1](../../picture/eformula1.png)

**基礎消費量の参考表 (baseCost = 512)**

| 加速率 | 基礎消費量 (AE) |
|------------|----------------|
| 2          | 256            |
| 4          | 1,024          |
| 8          | 2,048          |
| 16         | 8,192          |
| 32         | 16,384         |
| 64         | 65,536         |
| 128        | 131,072        |
| 256        | 524,288        |
| 512        | 268,435,456    |
| 1024       | 2,147,483,648 |

### エナジーカードによる電力軽減

エナジーカードを挿入すると、電力消費量が軽減されます。効果はカードの枚数に応じて増加しますが、追加効果はどんどん小さくなります。

定義: \( N \) = エナジーカードの数

**電力軽減の計算式**:  
![eformula1](../../picture/eformula2.png)

**電力軽減の参考表**:

| カードの枚数 | 効率 | 実際の電力軽減率 |
|----------------|------------|------------------|
| 0              | 0%         | 100%             |
| 1              | 10%        | 90%              |
| 2              | 14.5%      | 85.5%            |
| 3              | 17.15%     | 82.85%           |
| 4              | 19.0%      | 81.0%            |
| 5              | 20.21%     | 79.79%           |
| 6              | 21.15%     | 78.85%           |
| 7              | 21.9%      | 78.1%            |
| ≥8             | 50%        | 50%              |

### 最終的な電力消費量

**計算式**:

**消費 = 基礎消費量 × 軽減率**

**計算例**:

- 加速率: 64×
- エナジーカード: 3
- 基礎消費: 65,536 AE
- 電力軽減: 82.85%
- **電力消費**: 65,536 × 0.8285 ≈ 54,267 AE

> 実際の消費量は、各環境によって異なる場合があります。
