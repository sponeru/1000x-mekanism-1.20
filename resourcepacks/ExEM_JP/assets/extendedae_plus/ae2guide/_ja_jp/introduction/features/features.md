# 追加される機能の概要

## I. パターンプロバイダーの機能強化

特に言及がない限り、 **AE2の<ItemLink id="ae2:pattern_provider" />** と **ExtendedAEの<ItemLink id="expatternprovider:ex_pattern_provider" />** にのみ対応しています。

### 1. スマートブロッキングモード

AE2オリジナルのブロックモードを強化したモードです。同じレシピが連続して実行される場合、ブロックをしないことで時間の無駄を防ぎます。(AdvancedAEのパターンプロバイダーにも対応)

> AE2オリジナルのブロックモードを有効にする必要があります。

<br/>

### 2. スマートスケーリングモード

クラフト注文したとき、必要な素材の合計を自動的に計算し、一括で実行させます。これは、パターンのレシピを注文数分に手動で乗算して変更するのと同じ効果があります。(AdvancedAEのパターンプロバイダーにも対応)

設定できる項目:
- 最大乗算倍率
- 複数のプロバイダーに対する分散スケーリング(並行処理と分散効率を上昇させます)
- 最小乗算倍率

**例**:

パターン: 1 丸石 + 2 溶岩バケツ → 1 石
1000個の石を注文したとき、1000個の丸石と2000個の溶岩バケツを一括で出力します。

**追加設定**:

プロバイダーの左にある入力ボックスで、最大出力制限を設定できます。

例:

パターン: 1 丸石 + 2 溶岩バケツ → 1 石
64に設定すると、1000個の石を注文したとき、一度に32個の丸石と64個の溶岩バケツまで一括で出力するようになります。

有効なデータタイプ: アイテム、液体、ガス(Mekanism)

<br/>

### 3. 手動乗算ボタン

プロバイダーのGUIに乗数設定ボタンが追加され、一つ一つ編集することなくパターンの乗数を変更できるようになりました。

## II. AE2の強化

### 1. クイック操作の機能強化

- **中クリックでブロックを取得**: ブロックを中クリックすると、それがMEネットワークに存在したとき、自動でそのブロックを取得することができます。(ワイヤレスターミナルが必要です)
- **クラフトパターンの自動補充**: パターンエンコーディングターミナルを開いた時、MEネットワークから空のパターンが自動的に補充されます。

<br/>

### 2. 注文監視画面の機能強化

- **素材をシフト+左クリック**: クラフトしている機械のGUIを開きます。
- **素材をシフト+右クリック**: 対応するプロバイダーのGUIを開き、そのパターンをハイライトします。また、ワールド内でもそのプロバイダーがハイライトされます。
- **キャンセルをシフト+左クリック**: JEIのブックマークに不足している素材を追加します。

<br/>

### 3. パターンエンコーディングターミナルの機能強化

- **登録機能**: 登録ボタンで、パターンを対応する機械のパターンプロバイダーに直接登録します。JEIで取得したレシピは機械名を検出でき、一致するプロバイダーを自動的に検索します。自動検索にはマッピングテーブルが使われ、登録画面で追加できます。
- **クイック消去**: 登録画面で検索ボックスを右クリックすると、入力内容を消去できます。

<br/>

### 4. パターンの表示 & 名前の表示強化

- パターンアクセスターミナルやパターンプロバイダーでパターンの出力量を表示します。
- パターンプロバイダーのUIに名付けた名前を表示します。

<br/>

### 5. AEのパフォーマンス改善

- 新しいコンフィグ**Crafting Pause Threshold** (デフォルト: 100000)を追加します。クラフト計画中に計算を一時停止する頻度を調整し、大規模なクラフト計画の計算速度を大幅に改善します。

<br/>

### 6. クォーツカッターの機能拡充

- **AEのブロックに対してシフト+右クリック**: ブロック名をクリップボードにコピーします。GregTechの大型機械からレシピ名のコピーもできます。

## III. ExtendedAEの強化

### 1. パターンのクイック登録

- **組立マトリックス自動検出**: ネットワーク内に組立マトリックスが存在する場合、パターンエンコーディングターミナルで作成されたパターン(クラフト、鍛冶台、石切台)のパターンが自動的にアップロードされます。既存のパターンが検出されたときには返却されるため、同じパターンが登録されることはありません。(エンコードボタンをクリックするときにShiftキーを押す必要があります)

<br/>

### 2. 設定可能なコンフィグの拡充

- 拡張型パターンプロバイダーのスロット数をコンフィグで調整できるようになります。

<br/>

### 3. ターミナル操作の強化

- **検索ハイライト**: ExtendedAEのパターンアクセスターミナルに検索ハイライト機能を追加します。一致するものは虹色の枠線でハイライトされます。

---

## IV. JEIとの連携強化

### 1. 相互作用

- **JEIのアイテムをシフト右クリック**: MEネットワークに在庫があるとき、アイテムを取得できます。在庫がない時は注文画面に移動します。
> ワイヤレスターミナルが必要です。アクセサリスロットに装備していても機能します。

- **JEIのアイテムを中クリック**: ネットワークにクラフトパターンが存在するとき、注文画面に移動します。

<br/>

### 2. JEIとターミナルの同期

- **F キー**: JEIのアイテム名を、AE2の検索バーに入力できます。
- **優先パターン登録**: パターンを作成するとき、JEIのブックマークにあるレシピが優先されます。
