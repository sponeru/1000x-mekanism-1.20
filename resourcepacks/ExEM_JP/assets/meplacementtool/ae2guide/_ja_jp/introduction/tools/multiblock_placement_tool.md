# マルチブロック建築ツール

<ItemImage id="meplacementtool:multiblock_placement_tool" scale="4" />

マルチブロック建築ツールは、同じブロックを一括で設置できるツールです。同じブロックの面に自動で設置できるため、大規模な建築に最適です。言うなれば「Construction Wand」です。

## 機能

- **一括設置**:  1/8/64/256/1024 ブロックを一括で設置できます。
- **スマート設置**: 同じブロックの面にのみ設置します。
- **ネットワーク接続**: <ItemLink id="ae2:wireless_access_point" />を使ってMEネットワークに接続できます。
- **18個の設定スロット**: 最大18種類のブロックを設定でき、素早く切り替えられます。
- **取り消し**: 1つ前の設置を取り消すことができます。
- **メモリーカード対応**: オフハンドに<ItemLink id="ae2:memory_card" />を持っているとき、設置した部品に自動的に設定を反映します。
- **設定カード(Mekanism)対応**: オフハンドにMekanismの設定カードを持っているとき、設置した機械に自動的に設定を反映します。

## 使い方

- **空中に右クリック**: 設定画面を開く
- **地面に右クリック**: ブロックを設置
- **Gキー長押し**: ラジアルメニューを開く
- **ラジアルメニューで左クリック**: ブロックを選択/数を選択

![Radial Menu](../../assets/multiblock_placement_tool_radial.png)

- **Ctrl+左クリック**: 一つ前の設置を取り消し
